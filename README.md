# optahunt
OptaHunt is a modular, privacy-first cookie consent and preference system for Divhunt, built for GDPR, ePrivacy, and Google Consent Mode v2 compliance.



## v9.0.0 — Major Release

⚠️ Breaking change (opt-in only)

- New i18n system with external dictionaries
- Geo-based language detection
- Custom language dropdown
- No markup changes required
- v8.x remains fully supported
